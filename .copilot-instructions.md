# Instructions pour GitHub Copilot

Ce dépôt contient le code source complet de **JeuTaime**, une application Flutter de rencontres ludiques et émotionnelles.

## Objectif général
Copilot doit aider à développer, améliorer ou maintenir les modules du jeu.  
Le but n’est pas un simple “dating app”, mais une expérience sociale interactive.

## Modules clés à connaître
- **Lettres** : messages séquentiels avec système de relance.
- **Bars** : mini-salons (2H/2F) avec défis, quiz, interactions et mini-jeux.
- **Offrandes & Points** : système de récompenses et cadeaux virtuels.
- **Adoption** : parrainage entre anciens et nouveaux utilisateurs.
- **Boîte à souvenirs** : archive des lettres et interactions marquantes.

## Style et contraintes
- Code en **Dart / Flutter**
- Design doux, chaleureux, romantique.
- Les mini-jeux doivent rester courts, symboliques, et favorisant le lien émotionnel.

## Fichiers importants
- `lib/main.dart` → point d’entrée
- `lib/screens/` → écrans de navigation principaux
- `lib/models/` → structures de données
- `lib/services/` → logique et intégration Firebase
- `lib/widgets/` → éléments d’interface réutilisables

Copilot doit prioriser :
1. La lisibilité du code
2. L’expérience utilisateur immersive
3. Le respect de la logique de jeu déjà existante

## Exemples d’assistances attendues
- Ajouter un mini-jeu interactif dans un bar.
- Créer un système d’animation ou de score de groupe.
- Améliorer les transitions entre lettres et défis.
- Proposer des idées de magie, offrandes ou interactions originales.
